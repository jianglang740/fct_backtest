###1: 捕捉极值出现的时间
base = (returns < 0) ? stddev(returns, 20) : close;
Alpha1 = rank(ts_argmax(base^2, 5)) - 0.5;
逻辑：若收益为负则取波动率，否则取收盘价，平方后找过去5天最大值所在位置，排名后中心化。反映极端行情发生的时间点。

###2: 量价背离

逻辑：对数成交量变化排名与涨跌幅排名的负相关，捕捉量价背离（放量不涨或缩量下跌）。

###3: 开盘价与成交量的负相关

逻辑：过去10天开盘价排名与成交量排名负相关，度量“开盘即放量/缩量”的反转效应。

###4: 低位时间序列排名
Alpha4 = -1 * Ts_Rank(rank(low), 9);
逻辑：对过去9天的最低价截面排名再做时序排名，取负值，倾向做多近期低价排名上升的股票。

###5: 价格偏离与绝对排名惩罚
Alpha5 = rank((open - (sum(vwap,10)/10))) * (-1 * abs(rank((close - vwap))));
逻辑：开盘价偏离10日均价的程度，乘以收盘价偏离VWAP的绝对排名，捕捉双重价格异动。

###6: 开盘与成交量的负相关
Alpha6 = -1 * correlation(open, volume, 10);
逻辑：与#3类似，但用原始值而非排名，捕捉开盘价与量能的时序负相关。

###7: 条件动量反转（区分成交量）

else

逻辑：放量时，用7日价格变动绝对值的60日时序排名乘以方向符号；缩量时直接-1。

###8: 累积收益的滞后反转

###9: 三态价格变动（过滤震荡）

###10: #9的截面排名版本
